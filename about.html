~About

This website serves as a repository for documentation on introductory study of C++ programming. Here, you will find slides and discussions on object oriented programming, resource management, algorithm analysis, and data structures. The material reviewed on this website is paced and presented in the same order as the material covered in Data Abstraction and Problem Solving with C++: Walls and Mirrors, Seventh Edition, by Frank M. Carrano, Pearson, 2017. This website assumes the reader has beginner knowledge of C++ programming and its foundations such as basic pointer arithmetic, array manipulation, and C++ classes.
